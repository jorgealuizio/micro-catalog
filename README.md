## Rodar a aplicação

#### Rodar o RabbitMQ

Clone o projeto de configuração Docker do RabbitMQ neste [link](https://github.com/yuri-calabrez/rabbitmq-micro-catalog). Rode ```docker-compose up```.

#### Crie os containers com Docker

```bash
$ docker-compose up
```

#### Accesse no browser

```
http://localhost:3001
```
